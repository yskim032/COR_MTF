# DTX Web Converter

이 프로젝트는 기존 파이썬 기반의 DTX 변환기를 최신 웹 UI를 갖춘 웹 애플리케이션으로 포팅한 것입니다.

## 주요 기능
- **드래그 앤 드롭 및 파일 탐색기**: 여러 DTX 파일을 웹 UI를 통해 간편하게 업로드할 수 있습니다.
- **다중 파일 처리**: 한 번에 여러 개의 파일을 전송하고, 변환된 파일을 ZIP 형태로 다운로드합니다.
- **통계 기능**: 사용 횟수(변환된 파일 수)와 고유 IP별 방문 기록을 날짜별로 저장하고 통계 페이지에서 확인할 수 있습니다.
- **최신 라우팅 및 디자인**: 다크 모드를 기본으로 한 모던 UI가 적용되어 있습니다.

## 설치 및 실행 방법

1. Python 3.x 환경이 준비되어 있어야 합니다.
2. 저장소를 클론한 뒤 프로젝트 루트 폴더에서 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
3. Flask 애플리케이션을 실행합니다:
   ```bash
   python app.py
   ```
4. 웹 브라우저에서 `http://localhost:5000` 로 접속하여 사용합니다.

## 배포 (Deployment)
이 애플리케이션은 Python Anywhere, Heroku, AWS EC2, 또는 로컬 서버 환경에 쉽게 배포할 수 있습니다. 
내장된 `app.py`를 실행하기만 하면 로컬 SQLite 데이터베이스를 생성하여 동작하므로 특별한 데이터베이스 환경설정이 필요하지 않습니다.

* Vercel 배포의 경우, Flask API 배포 설정(`vercel.json`)을 추가하고 DB를 외부 저장소(PostgreSQL 등)로 변경해야 영구적인 통계 기록 관리가 가능합니다.
